# install-setup

This script installs my working environment on Debian.
Tested only on Debian 11.

## Usage

```
Arguments :
        -h : Display usage.
        -d : Setup Debian desktop.
        -v : Setup vim.
        -z : Setup zsh.
        -t : Setup tmux.
        -g : Setup golang env.
```

## Install

```bash
$ ./setup.sh
```
